# 🔐 AI Password Generator

A modern and responsive AI Password Generator built with HTML, CSS, and JavaScript. This tool allows users to generate highly secure passwords, analyze password strength in real time, and copy passwords instantly with a single click.

---

## 🚀 Features

- Generate strong random passwords
- Real-time password strength analysis
- Security score calculation (0–100)
- Detects:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Copy password with one click
- Modern dark UI design
- Fully responsive layout
- Fast and lightweight

## ⚡ How It Works

### Generate Password

Click the **Generate Strong Password** button to create a highly secure password containing:

- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

### Analyze Password

Enter your own password and the tool will automatically:

- Calculate a security score
- Determine password strength
- Display detailed security feedback

### Copy Password

Use the **Copy Password** button to instantly copy the password to your clipboard.

## 🎯 Strength Levels

| Score | Strength |
|---------|---------|
| 0 - 39 | Weak |
| 40 - 69 | Medium |
| 70 - 89 | Strong |
| 90 - 100 | Very Strong |
